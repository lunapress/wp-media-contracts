# WP Media Wrapper Contracts

Contracts and wrappers for WordPress Media and Attachments API.

## Supported Functions

- `wp_handle_upload()`
- `media_handle_upload()`
- `wp_get_attachment_url()`
- `wp_insert_attachment()`
- `wp_generate_attachment_metadata()`
- `get_attached_file()`
- `update_attached_file()`
